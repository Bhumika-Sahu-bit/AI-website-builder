# 🤖 AI Website Builder

AI Website Builder is an intelligent full-stack web application that generates complete, production-ready websites from a simple text prompt. Just describe the website you want — the AI understands your requirements and instantly writes all the code, creates all the files, and gives you a live preview right in the browser. No coding knowledge required.

## ✨ Features

- 🧠 **AI Code Generation** — Generates a complete website from just a text description
- ⚡ **Dual Mode** — Supports both HTML/CSS/JS and React + Vite + Tailwind CSS projects
- 👁️ **Live Preview** — Instantly preview the generated website inside the browser
- 📁 **File Explorer** — Browse all generated files and folders in a clean sidebar
- ✏️ **AI File Editor** — Select any file and ask AI to update or improve it
- 📦 **Download as ZIP** — Download the entire project with one click and run it locally
- 🔄 **Smart Detection** — Automatically detects whether to generate HTML or React based on your prompt

## 🚀 How to Use

Using AI Website Builder is very simple — just follow these steps:

**Step 1 — Open the Website**
Go to the website in your browser. You will see a prompt input box on the home page.

**Step 2 — Write Your Prompt**
Describe the website you want to build. Be as detailed as possible for better results. Mention the type of website, sections, colors, and features you want.

**Step 3 — Choose the Project Type**
Specify in your prompt whether you want a plain HTML/CSS/JS website or a React + Vite + Tailwind project.

**Step 4 — Click Generate**
Hit the Generate button and wait a few seconds. The AI will generate all the files and folders for your project automatically.

**Step 5 — Preview Your Website**
Once generated, a live preview of your website appears instantly. You can see exactly how it looks in the browser.

**Step 6 — Browse & Edit Files**
Open the file explorer to see all generated files. Click on any file to view its code. If you want to change something, use the AI editor — just describe what you want to change and the AI will update the file for you.

**Step 7 — Download Your Project**
When you are happy with the result, click the Download button to get the full project as a ZIP file. Extract it, run `npm install` and `npm run dev`, and your website is ready locally.
